Ajuda

Esta seção servirá para que você compreenda basicamente esse sistema.

1 - O que é o Cubo?

É um equipamento com quatro telas (televisões) controladas por meio de pequenos computadores que obedecem a este painel de controle. Esse cubo estará na sua escola e você poderá utilizá-lo.


2 - Por onde você começa?

Primeiro, compreenda basicamente os botões "Conteúdos" e "Telas".


Conteúdos:
    - Conteúdos é o local no sistema onde você acrescenta vídeos e/ou imagens que deseja mostrar à sua turma.
    - Para mostrá-los de maneira organizada, você precisará criar uma Playlist (lista de reprodução) e atribuir os conteúdos inseridos à essa Playlist.
    - São as Playlists que você vai associar aos Grupos de Telas.

Telas:
    - Telas é onde você organiza os Grupos de Telas (televisões).
    - Cada Grupo de Telas mostra uma Playlist diferente. Para isso, é preciso associar a Playlist desejada ao Grupo de Telas.
    - Existem quatro televisões no Cubo. Você pode associá-las de diferentes formas.         - Sugestão de uso 1: A mesma Playlist em todas as telas.
        - Sugestão de uso 2: Alternar duas Playlists diferentes.

Como cadastrar conteúdo

Para cadastrar um conteúdo, você deve clicar em "Conteúdos" no menu superior direito, em seguida clique em "Adicionar".

Clique sobre a imagem para vê-la no tamanho real

Uma nova janela vai se abrir para que você busque no seu computador pelo conteúdo a ser adicionado. Escolha o conteúdo e prossiga.

Clique sobre a imagem para vê-la no tamanho real

Então uma nova janela se abrirá para ajudar você a organizar melhor o conteúdo inserido. Nessa janela, você poderá criar uma nova Etiqueta para identificar o conteúdo ou selecionar uma etiqueta já criada.

Clique sobre a imagem para vê-la no tamanho real

Caso queira adicionar uma nova etiqueta, digite o nome da nova etiqueta no campo "Nova etiqueta", em seguida clique em "Adicionar etiqueta".

Clique sobre a imagem para vê-la no tamanho real

Você verá a nova etiqueta ser adicionada na lista de etiquetas já selecionada. Então clique em "Continue".

Clique sobre a imagem para vê-la no tamanho real

Finalmente, basta clicar em OK após a tela de aviso e seu conteúdo já estará incluído.
